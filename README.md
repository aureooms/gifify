# gifify

> Convert videos to GIFs.

[![AUR package](https://img.shields.io/aur/version/gifify)](https://aur.archlinux.org/packages/gifify)


## Dependencies

  - ffmpeg
  - gifsicle
  - convert
  - gifski

## Install

```sh
make DESTDIR=/ PREFIX=/usr install
```

## Usage

```sh
> gifify -i video.mp4 -o video.gif
```

## Manual

```sh
> gifify -h
```
